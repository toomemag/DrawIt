# Build process for APK
1. In Android Studio navigated to `Build -> Generate Signed App Bundle or APK...`
2. Selected APK
3. Created a key by following [this](https://developer.android.com/studio/publish/app-signing)
4. Selected `Next`
5. Selected build variant as `release`
6. Selected `Create`
